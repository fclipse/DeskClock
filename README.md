탁상시계 프로젝트
=================

* 올렸다가 내려놓으면 자동으로 타이머나 스톱워치가 실행되는 방식
 
* 기능 목록
1. 제대로 올린 상태에서 올렸다 내리면 스톱워치 시작
2. 나머지 각 면마다 5분 타이머, 30분 타이며, 50분 타이머가 있음.(밑면 제외)
3. 위에 버튼 누르면 멈춤.
4. 화면을 위로 해서 들었다가 놓으면 멈춤 + 리셋.
